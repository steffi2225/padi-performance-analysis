# 📊 PaDi Marketplace Analysis

## 📌 Overview
This project analyzes transaction data from the PaDi platform to understand seller and buyer activity, revenue trends, and platform performance.

## 🛠️ Tools Used
- SQL
- Tableau
- Excel

## 📈 Key Insight
- Large gap between registered and active sellers
- Revenue concentrated in small group of sellers

## 💡 Recommendation
- Improve seller onboarding process
- Increase activation strategy
